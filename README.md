
# Web Application for a Real Estate Agency
Created in Angular and Node.js

## What to install: <br>
```
npm install multer
npm install ng2-charts@2.4.2 --save 
npm install chart.js --save 
ng add @angular/material 
```

## To open the app:


  backend:
  ```
  cd backend 
  tsc
  npm run serve
  ```
  
  frontend:
  ``` 
  cd frontend/app 
  ng serve
  ```
